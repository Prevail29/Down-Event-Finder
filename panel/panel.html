CharacterData Filter

Filters allow for fine-tuning the observable results by determining whether a mutation will receive the state "warning" or "problem":

  • If a filter is checked, mutations will receive the state "warning".
  • If a filter is unchecked, mutations will receive the state "problem".

Changes to the filters need to be applied before the test is run!
The default options are recommended, and changes should only be made in very specific circumstances.
