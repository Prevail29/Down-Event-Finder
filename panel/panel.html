Filters allow for fine-tuning the observable results:

  • If a filter is checked, the selected option will not be observed.
  • If a filter is unchecked, the selected option will be observed and will appear in the results area.

Changes to the filters need to be applied before the test is run!
The default options are recommended, and changes should only be made in very specific circumstances.
